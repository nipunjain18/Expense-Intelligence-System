# Expense Intelligence System

## Project Goal

Build a Personal Finance Management System using Python to learn:

* Real software development
* Problem solving
* File handling
* JSON storage
* Project structure
* Git & GitHub
* Software design

---

## Version 1 Scope

Single User System

The application is designed for one user.

The user can manage:

* Multiple accounts
* Income
* Expenses
* Transfers
* Debt tracking
* Categories
* Dashboard

---

## Features

### Account Management

User can:

* Add account
* View accounts
* Update account balance

Examples:

* Cash
* UPI
* SBI Bank
* Investment Account
* Credit Card

---

### Category Management

User can:

* Add category
* View categories

Default examples:

* Food
* Shopping
* Transport
* Gym
* Subscription
* Healthcare
* Skincare

User can create custom categories.

---

### Transaction Management

User can:

* Add income
* Add expense
* View transactions

Transaction types:

* Income
* Expense

Examples:

Income:

* Pocket Money
* Salary
* Freelance

Expense:

* Food
* Shopping
* Gym

---

### Transfer Management

Transfer money between accounts.

Example:

SBI Bank → Investment Account

₹5000

Transfer should not be counted as an expense.

It only moves money between accounts.

---

### Debt Tracking

Track money lent and borrowed.

Money Lent Example:

Rahul owes ₹1000

Money Borrowed Example:

Amit lent me ₹5000

Status:

* Pending
* Paid

---

### Dashboard

Show:

* Total Income
* Total Expense
* Net Savings
* Account Balances
* Pending Debts
* Pending Loans
